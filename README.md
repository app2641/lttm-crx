lttm-crx
========

Looks Tiqav To Misawa (useful Chrome Extension)


- LGTM: `!l or !ll...`
- tiqav: `!t<query>`
- misawa: `!m<query>`

Select image with ↑/↓(Ctrl + p/n), then hit Enter or Tab to complete

[![LTTM](http://img.youtube.com/vi/jFr1A2a_eeI/0.jpg)](http://www.youtube.com/watch?v=jFr1A2a_eeI)
